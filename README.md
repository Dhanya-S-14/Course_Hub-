# 🎓 CourseHub

> Your Complete Job Preparation Portal - From Courses to Placements!



---

## 📌 Overview

**CourseHub** is a comprehensive job preparation portal that helps students prepare for placements at both **Service-Based** and **Product-Based** companies. It provides:

- 📚 Free course resources for various technologies
- 🏢 Company-specific preparation guides
- 💻 DSA patterns and coding questions
- 📝 Aptitude & interview preparation resources

---

## ✨ Features

### 🚀 For Job Seekers
- **Company Prep Pages** - Dedicated guides for TCS, Infosys, Wipro, Google, Amazon, Meta, Microsoft, Apple, Adobe & more
- **DSA Patterns** - Company-wise coding patterns and must-do problems
- **Aptitude Focus** - Topics and resources for quantitative aptitude
- **Interview Tips** - Real insights from candidates

### 📖 Course Resources
- Curated free courses from Coursera, Udemy, NPTEL, etc.
- Courses organized by category (Data Science, Web Dev, Cloud, AI/ML)
- Company-specific course recommendations

### 🎯 Company Categories

#### Service-Based Companies
| Company | Difficulty | Focus Areas |
|---------|------------|-------------|
| TCS | Easy | Arrays, Strings, Aptitude |
| Wipro | Easy | Arrays, Sorting, Aptitude |
| Infosys | Easy-Medium | Pseudo Code, Arrays |
| HCLTech | Easy-Medium | Technical MCQ, Arrays |
| Cognizant | Easy | Aptitude, Reasoning |
| Accenture | Easy-Medium | Communication, Coding |

#### Product-Based Companies
| Company | Difficulty | Focus Areas |
|---------|------------|-------------|
| Google | Hard | DP, Graphs, Trees |
| Amazon | Medium-Hard | Sliding Window, BST |
| Meta | Hard | Arrays, Two Pointers |
| Microsoft | Medium | Trees, Graphs |
| Adobe | Medium | Binary Search, Sorting |
| Apple | Hard | Fundamentals, System Design |

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- MongoDB (local or Atlas)

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/coursehub.git
cd coursehub

# Install dependencies
npm install

# Start the server
npm start
```

Visit: http://localhost:5000

## 📖 Preparation Roadmap

### 🟢 Phase 1: Foundation (Week 1-2)
- [ ] Learn one programming language (C++/Java/Python)
- [ ] Understand Arrays and Strings
- [ ] Practice basic problem solving

### 🟡 Phase 2: Core Skills (Week 3-4)
- [ ] Data Structures (Arrays, Linked Lists, Trees)
- [ ] Algorithms (Sorting, Searching)
- [ ] Start aptitude preparation

### 🟠 Phase 3: Advanced (Week 5-6)
- [ ] Dynamic Programming
- [ ] Graphs and Trees
- [ ] Company-specific preparation

### 🔴 Phase 4: Interview Ready (Week 7-8)
- [ ] Mock interviews
- [ ] Resume building
- [ ] Soft skills & communication
